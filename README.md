## Installation

### Standalone Installation
```bash
git clone https://github.com/tridentbiz/adios-model-performance-monitoring.git
cd adios-model-performance-monitoring
./install.sh
```

### AdiOS Integration
```bash
cd /path/to/0001-adios
git submodule add https://github.com/tridentbiz/adios-model-performance-monitoring.git plugins/model-performance-monitoring
adios plugin install model-performance-monitoring
```
